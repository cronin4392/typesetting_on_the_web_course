# Shortcuts #

⌘: command

⌥: option / alt

⌃: shift

| Description                              | Shortcut |
| ---------------------------------------- | -------- |
| Move to beginning of line                | ⌘ ←      |
| Move to end of line                      | ⌘ →      |
| Move to previous word                    | ⌥ ←      |
| Move to next word                        | ⌥ →      |
| Screenshot / Measure tool                | ⌘ ⌃ 4    |
| Change between applications              | ⌘ tab    |
| Change between windows in an application | ⌘ `      |
